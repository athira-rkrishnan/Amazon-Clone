# Amazon Clone Website
This project is a responsive front-end clone of the Amazon e-commerce website. Built with HTML, CSS, and JavaScript, it demonstrates modern web development techniques, including interactive sliders, dropdown menus, and a navigation sidebar. The project features multiple product carousels implemented using Swiper.js and mimics key elements of Amazon's user interface. 

## Features 
* **Responsive Design:** Fully adapts to mobile, tablet, and desktop screens.
* **Header & Navigation:** Includes location, language, currency options, and user account menus.
* **Sidebar Navigation:** Collapsible menu for categories and services.
* **Image Slider:** Eye-catching promotional banners with manual navigation.
* **Product Carousels:** Multiple horizontally scrollable sections showcasing bestsellers, deals and categories.
* **Back to Top Button:** Smooth scroll back to top of the page.
* **Footer Sections:** Informative links, policies and regional settings.

## Technologies Used
* HTML5
* CSS3
* JavaScript 
* Swiper.js for carousels
* Font Awesome for icons
